Building an Application

Tutorial Data

Introduction

NoesisGUI provides a powerful framework for building multi-platform applications. Applications created this way are fully portable. This tutorial is dedicated to creating an application using this framework. In case you are not interested in this abstraction and you want full control of your application and its window you probably should read the integration tutorial.

The main classes for building an application are the class Application and the class Window.

Window

A Window is used as a container that shows its contents in an operating system window.

<Window
  xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
  xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
  Background="White" Title="RSS Reader" Width="285" Height="385" Icon="rss.ico">

    <DockPanel Background="#FF2F3F4F" HorizontalAlignment="Center" VerticalAlignment="Center">
        <TextBlock DockPanel.Dock="Top" Text="RSS Reader" FontSize="32" Foreground="White"
          TextAlignment="Center" Margin="0,10,0,0"/>
        <Border DockPanel.Dock="Top" Background="SlateGray" BorderBrush="Gray"
          BorderThickness="1" CornerRadius="2" Margin="10" Padding="5">
            <StackPanel Orientation="Horizontal">
                <TextBlock Text="URL: " VerticalAlignment="Center"/>
                <TextBox x:Name="Address" Width="180" Text="http://www.metacritic.com/"/>
                <Button Content="Go" Click="OnGoToClicked" Margin="2,0,0,0"/>
            </StackPanel>
        </Border>
        <Grid DockPanel.Dock="Bottom" Margin="10">
            <Grid.ColumnDefinitions>
                <ColumnDefinition/>
                <ColumnDefinition/>
            </Grid.ColumnDefinitions>
            <Button Grid.Column="0" Content="Prev" Click="OnPrevClicked"/>
            <Button Grid.Column="1" Content="Next" Click="OnNextClicked"/>
        </Grid>
        <Border Background="SlateGray" BorderBrush="Gray" BorderThickness="1"
          CornerRadius="2" Margin="10,0" Padding="5">
            <StackPanel x:Name="ContentPanel">
                <TextBlock x:Name="Title" FontSize="16" FontWeight="Bold"
                  Foreground="LightSkyBlue" TextAlignment="Center" Margin="0,0,0,10"/>
                <ScrollViewer VerticalScrollBarVisibility="Auto" Height="150">
                    <TextBlock x:Name="Desc" TextWrapping="Wrap" Width="200"/>
                </ScrollViewer>
            </StackPanel>
        </Border>
    </DockPanel>
</Window>

Application

The application is implemented in a separate .xaml file, normally called app.xaml, that indicates the main window and global resources.

<Application
  xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
  xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
  StartupUri="RssReader.xaml">

<Application.Resources>
    <ResourceDictionary Source="../../../../Gui/Themes/Content/NoesisStyle.xaml"/>
</Application.Resources>

</Application>

Code-Behind

The behavior of the application is implementing using a code-behind class. This is a native class that is referenced in the XAML using x:Class. In this sample, we are going to implement the events OnGoToClicked, OnPrevClicked and OnNextClicked. Changes in the window XAML are highlighted.

 <Window
   xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
   xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
   Background="White" Title="RSS Reader" Width="285" Height="385" Icon="rss.ico"
   x:Class="RssReader">

     <DockPanel Background="#FF2F3F4F" HorizontalAlignment="Center" VerticalAlignment="Center">
         <TextBlock DockPanel.Dock="Top" Text="RSS Reader" FontSize="32" Foreground="White"
           TextAlignment="Center" Margin="0,10,0,0"/>
         <Border DockPanel.Dock="Top" Background="SlateGray" BorderBrush="Gray"
           BorderThickness="1" CornerRadius="2" Margin="10" Padding="5">
             <StackPanel Orientation="Horizontal">
                 <TextBlock Text="URL: " VerticalAlignment="Center"/>
                 <TextBox x:Name="Address" Width="180" Text="http://www.metacritic.com/"/>
                 <Button x:Name="GoTo" Content="Go" Click="OnGoToClicked" Margin="2,0,0,0"/>
             </StackPanel>
         </Border>
         <Grid DockPanel.Dock="Bottom" Margin="10">
             <Grid.ColumnDefinitions>
                 <ColumnDefinition/>
                 <ColumnDefinition/>
             </Grid.ColumnDefinitions>
             <Button x:Name="Prev" Grid.Column="0" Content="Prev" Click="OnPrevClicked"/>
             <Button x:Name="Next" Grid.Column="1" Content="Next" Click="OnNextClicked"/>
         </Grid>
         <Border Background="SlateGray" BorderBrush="Gray" BorderThickness="1"
           CornerRadius="2" Margin="10,0" Padding="5">
             <StackPanel x:Name="ContentPanel">
                 <TextBlock x:Name="Title" FontSize="16" FontWeight="Bold"

                 <ScrollViewer VerticalScrollBarVisibility="Auto" Height="150">
                     <TextBlock x:Name="Desc" TextWrapping="Wrap" Width="200"/>

         </Border>
     </DockPanel>
 </Window>

The code-behind class is a class implemented in C++ following the guidelines described in the Extending NoesisGUI tutorial.

#pragma warning(disable: 4275)
#pragma warning(disable: 4251)


#include <Noesis.h>
#include <NsGui/Window.h>
#include <NsCore/ReflectionImplement.h>
#include <NsCore/TypeId.h>
#include <NsCore/Package.h>
#include <NsGui/RoutedEvent.h>
#include <NsGui/TextBlock.h>


using namespace Noesis;
using namespace Noesis::Core;
using namespace Noesis::Gui;


const NsChar* const gTitles[] =
{
    "Monkey Island 1",
    "Monkey Island 2",
    "Monkey Island 3"
};

const NsChar* const gBodies[] =
{
    "The Secret of Monkey Island is a 1990 point-and-click graphic adventure game developed and published by Lucasfilm Games. It takes place in a fantastical version of the Caribbean during the age of piracy. The player assumes the role of Guybrush Threepwood, a young man who dreams of becoming a pirate and explores fictional islands while solving puzzles",
    "Monkey Island 2: LeChuck's Revenge is an adventure game developed and published by LucasArts in 1991. It was the second game of the Monkey Island series, following The Secret of Monkey Island, and the sixth LucasArts game to use the SCUMM engine. It was the first game to use the iMUSE sound system.",
    "The Curse of Monkey Island is an adventure game developed and published by LucasArts, and the third game in the Monkey Island series. It was released in 1997 and followed the successful games The Secret of Monkey Island and Monkey Island 2: LeChuck's Revenge. The game is the twelfth and last LucasArts game to use the SCUMM engine, which was extensively upgraded for its last outing before being replaced by the GrimE engine for the next game in the series, Escape from Monkey Island. The Curse of Monkey Island is the first Monkey Island game to include voice acting, and has a more cartoon-ish graphic style than the earlier games"
};

////////////////////////////////////////////////////////////////////////////////////////////////////
class RssReader: public Window
{
public:
    void OnInit()
    {
        mIndex = 0;
        ParentClass::OnInit();
        Update();
    }

    void OnGoToClicked(BaseComponent* sender, const RoutedEventArgs& e)
    {
    }

    void OnPrevClicked(BaseComponent* sender, const RoutedEventArgs& e)
    {
        mIndex = mIndex == 0 ? 2 : mIndex -  1;
        Update();
    }

    void OnNextClicked(BaseComponent* sender, const RoutedEventArgs& e)
    {
        mIndex = mIndex == 2 ? 0 : mIndex +  1;
        Update();
    }

private:
    void Update()
    {
        TextBlock* title = NsStaticCast<TextBlock*>(FindName("Title"));
        title->SetText(gTitles[mIndex]);

        TextBlock* desc = NsStaticCast<TextBlock*>(FindName("Desc"));
        desc->SetText(gBodies[mIndex]);
    }

private:
    NsUInt mIndex;

    NS_IMPLEMENT_INLINE_REFLECTION(RssReader, Window)
    {
        NsMeta<TypeId>("RssReader");
        NsFunc("OnGoToClicked", &RssReader::OnGoToClicked);
        NsFunc("OnPrevClicked", &RssReader::OnPrevClicked);
        NsFunc("OnNextClicked", &RssReader::OnNextClicked);
    }
};

////////////////////////////////////////////////////////////////////////////////////////////////////
extern "C" NS_DLL_EXPORT
void NsRegisterReflection(ComponentFactory* factory, NsBool registerComponents)
{
    NS_REGISTER_COMPONENT(RssReader)
}

Building Data Resources

You must build the resources used by your application as described in the BuildTool documentation. For this tutorial makes sure that the package Gui/Tutorials is properly built.

Running the Application

You have two options for launching the application:

  1. The easy one is using the binary Gui.Launcher.exe that comes with the SDK.
> Gui.Launcher.exe Gui/Tutorials/Application/app.xaml
  1. The second option is using the API provided by NoesisGUI. This way you generate the executable, although the entry point and main loop are controlled by Noesis. You need to add the following file in your project to initialize the application and enter the main loop:
#pragma warning(disable: 4275)
#pragma warning(disable: 4251)


#include <Noesis.h>
#include <NsGui/NoesisApplication.h>
#include <NsGui/CommandLine.h>
#include <NsGui/EntryPoint.h>


using namespace Noesis;
using namespace Noesis::Core;


////////////////////////////////////////////////////////////////////////////////////////////////////
namespace
{
struct LauncherApplication: public Noesis::Gui::NoesisApplication
{
    /// From BaseApplication
    //@{
    const NsChar* GetApplicationName() const
    {
        return "Launcher";
    }
    //@}
};
}

////////////////////////////////////////////////////////////////////////////////////////////////////
int NsMain(int argc, char **argv)
{
    Noesis::Gui::CommandLine commandLine(argc, argv);

    LauncherApplication theApp;
    theApp.SetApplicationFile("Gui/Tutorials/Application/app.xaml");

    if (theApp.Init(commandLine))
    {
        theApp.Run();
        theApp.Shutdown();
    }

    return 0;
}
